How to define a table?
Each table cell is defined by a <td> and a </td> tag. td stands for table data. Everything between <td> and </td> is the content of a table cell.
<tr> tag
The <tr> tag defines a row in an HTML table. A <tr> element contains one or more <th> or <td> elements.
<th> tag
The <th> tag defines a header cell in an HTML table. It contains header information. The text in <th> elements are bold and centered by default.
<td> tag
The <td> tag defines a standard data cell in an HTML table. It contains data. The text in <td> elements are regular and left-aligned by default.
<thead> tag
The <thead> tag is used to group header content in an HTML table. The <thead> element is used in conjunction with the <tbody> and <tfoot> elements to specify each part of a table (header, body, footer). It must have one or more <tr> tags inside.
<tfoot> tag
The <tfoot> tag is used to group footer content in an HTML table. The <tfoot> element is used in conjunction with the <thead> and <tbody> elements to specify each part of a table (footer, header, body). It must have one or more <tr> tags inside.
<caption> tag
The <caption> tag defines a table caption. The <caption> tag must be inserted immediately after the <table> tag.
What is the difference between colspan and rowspan?
colspan merges cells horizontally across columns, while rowspan merges cells vertically across rows. Both are HTML table attributes used to make a single cell span the space of multiple cells for a more organized layout.
What is the difference between cells spacing and padding?
Cell spacing is the space between adjacent cells, while cell padding is the space inside a single cell, between its content and its border.
How do you define table borders using CSS?
The CSS border property is used to specify the width, style, and color of table borders. It could be defined by table, and curly brackets, and border property inside. The border property is a shorthand property for:
  • border-width - sets the width of the border
  • border-style - sets the style of the border (required)
  • border-color - sets the color of the border
How do you control border widths using CSS?
Border width in CSS is controlled using the border-width property or its individual longhand properties: border-top-width, border-right-width, border-bottom-width, and border-left-width.
